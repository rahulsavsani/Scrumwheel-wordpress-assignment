== Description ==

UT Elementor Addons Lite provides Elementor addons. Enhance your Elementor page building experience with creative elements and extensions. Add powers to your page builder using our easy-to-use elements those were designed to make your next WordPress website build easy. Each element comes with bunch of options to control every possible things. You can achieve nearly any design with your imagination.

== Installation ==
1. Upload `ut-elementor-addons-lite` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy !!!
