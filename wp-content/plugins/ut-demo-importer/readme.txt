== Description ==

UT Demo Importer is a plugin to import demo of uncodethemes.


== Installation ==
1. Upload `ut-demo-importer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy !!!

* English


= Features =
* <strong>Import Demo of UncodeThemes.</strong>
